## Descripción del Proyecto

El objetivo principal de este proyecto es servir como un campo de pruebas para aprender, implementar y experimentar con diferentes tecnologías, patrones de diseño y buenas prácticas en el ecosistema de Android. [5] No pretende ser una aplicación completamente funcional para el usuario final, sino más bien una colección de ejemplos y pruebas de concepto. Puede contener errores o bugs en el proceso ya que es una aplicación que ha sufrido muchos cambios en el proceso [4]

### Conceptos que se están probando

*   **Lenguaje:** [Kotlin ]
*   **Arquitectura:** [MVVM | MVI | Clean Architecture]
*   **Inyección de Dependencias:** [ Koin]
*   **Programación Asíncrona:** [Coroutines + Flow ]
*   **Networking:** [Retrofit + OkHttp]
*   **Base de Datos Local:** [Room ]
*   **Componentes de Jetpack:**
    *   [Compose | Vistas (XML)]
    *   Navigation Component
    *   Lifecycle
    *   LiveData | StateFlow
    *   ViewModel
*   **Pruebas (Testing):**
    *   Pendiente
